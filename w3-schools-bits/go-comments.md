### GO COMMENTS

A **GO** comment is a text that is ignored upon exection. Comments make the code more readable and it is used to explain the code. It prevent code exection when testing an alternative code and **GO** supports single-line or multi-line comments.

## Go Single-line Comments

Single line comments start with two forward slashes `(//)`. Any text between the slashes are ignored by the compiler.

## Go Multi-Line Comments

Multi-line comments start with `(/*)` and ends with `(*/)`. Any text between it is ignored by the compiler.

You can used comments to prevent code from being exected.
